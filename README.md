# e-plantShopping

## Aperçu du projet
Ce projet est une plateforme de vente de plantes en ligne, permettant aux utilisateurs de parcourir et d'acheter une variété de plantes. L'application offre une interface conviviale et des fonctionnalités telles que la gestion du panier, la visualisation des produits, et un système de gestion des commandes.
